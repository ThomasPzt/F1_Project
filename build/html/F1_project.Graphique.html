class GraphiqueClassement¶

Bases: object

Cette classe fournit des fonctionnalités pour afficher le classement des pilotes par tour et les temps prédits par tour avec une couleur différente pour un pilote spécifique.

Attributes: - liste_pilotes (list): Une liste des numéros des pilotes. - dico_pilotes (dict): Un dictionnaire contenant les correspondances entre les noms des pilotes et leurs numéros.

Methods: - afficher_classement(df_resultat): Affiche le classement des pilotes par tour. - afficher_temps_predit(df_resultat, pilote, stand): Affiche le temps prédit par tour avec une couleur différente

pour un pilote spécifique.

static afficher_temps_predit(df_resultat, pilote, stand)¶

Affiche le temps prédit par tour avec une couleur différente pour un pilote spécifique.
